Operator Guide

How to use each workflow and what to expect from the labels.

This guide keeps the command center approachable for demos, submission reviews, and day-to-day testing.

01

Analyze text

Paste a message, run analysis, then review label, confidence bars, and keyword signals before taking action.

02

Inspect screenshots

Upload clear screenshots so OCR extracts better text before the classifier evaluates the content.

03

Read the dashboard

Use the analytics panel to understand session volume, threat ratio, and confidence trends over time.

04

Scan Gmail carefully

Authenticate with a Gmail App Password, scan the inbox, and only select fake messages if you are confident they should be deleted.

05

Retrain with good data

Provide balanced, labeled message data so the model improves rather than overfitting to one label type.

06

Use human judgment

Suspicious output means review manually. Real output is lower risk, not a guarantee of safety.

Label Guide

How the probability thresholds map to labels

Real

Fake probability under 40%. The wording looks more like legitimate communication.

Suspicious

Fake probability between 40% and 60%. Pause and review before clicking or replying.

Fake

Fake probability above 60%. Treat links, payment requests, and account warnings as unsafe until verified.
